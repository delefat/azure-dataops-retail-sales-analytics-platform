\# Azure DataOps Retail Sales Analytics Platform



This project implements an end-to-end Azure Data Engineering platform using:



\- Azure Data Lake Storage Gen2

\- Azure Databricks

\- Unity Catalog

\- Azure Synapse Analytics

\- Power BI

\- Azure Data Factory

\- Bicep

\- GitHub Actions

\- Azure DevOps

\- Testing and monitoring



The platform processes the Online Retail II dataset through Bronze, Silver and Gold layers before serving curated analytics through Synapse and Power BI.
